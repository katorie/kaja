# Welcome Ruby Kaja

Ruby Kajaは、RubyやRubyコミュニティに貢献している人達に与えられる称号です。
WEBサイトは http://kaja.rubyist.net/ です。

## 動かし方
```
$ gem install bundler
$ git clone git://github.com/tyabe/kaja.git
$ cd kaja
$ bundle install --without production
$ padrino s
```
